# Global Fuel Price Analysis (1970–2026)

---

# Executive Summary

This project analyzes global fuel price trends from 1970 to 2026 using Python-based data analysis and visualization techniques. The objective is to identify long-term fuel price patterns, detect market fluctuations, and forecast future fuel price behavior using machine learning models.

The project demonstrates data cleaning, exploratory data analysis (EDA), trend visualization, and predictive modeling using historical fuel price data.

---

# Business Problem

Fuel prices directly impact:

- Transportation industries
- Global inflation
- Supply chain costs
- Government policies
- Consumer spending

Businesses and policymakers require fuel price forecasting to:

- Reduce operational risks
- Optimize logistics planning
- Predict future market behavior
- Improve financial decision-making

This project helps analyze historical trends and forecast future fuel price movements for strategic planning.

---

# Methodology

The following steps were performed in this project:

## 1. Data Collection
- Collected historical global fuel price dataset from 1970–2026.

## 2. Data Cleaning
- Removed missing values
- Handled duplicates
- Converted date columns
- Standardized numerical values

## 3. Exploratory Data Analysis (EDA)
- Trend analysis
- Distribution analysis
- Correlation analysis
- Outlier detection

## 4. Data Visualization
Created:
- Line charts
- Trend graphs
- Moving average plots
- Forecast visualizations

## 5. Forecasting Model
Implemented Machine Learning models using:
- Linear Regression
- Time Series Forecasting

## 6. Future Prediction
Predicted fuel prices for upcoming years based on historical trends.
